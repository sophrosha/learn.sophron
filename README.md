# Learn.Sophron
## Основная информация
- Продукт по созданию курсов языков программирования для последующего изучения.
- Данный продукт учит пользователя языку программирования. Следит за ним и дает оценивание.
### Особенности:
```
- Собственная система оценивания(отличная от duolingo, др)
- Работа вместе с Gitea(Создание аккаунта после регистрации, опубликование и хранение кода после заданий)
- Создание пользовательских задач(для других)
- Развертка песочницы в docker на определенных видов курсов.
- Режим гостя и интеграция системы авторизации NeoID
- Система АнтиДебил
- IDE песочница встроенная не только в курсах
```
## Какие будут использованны инструменты/ЯП
```
- ЯП: python, JS, html, css
- Фреймворки: NodeJS, VueJS
- Контейнеризация: Docker
- Библиотеки: Pyodide
- СУБД: MongoDB
- Git: Gitea
```
## FAQ
- Что за система АнтиДебил?
```
АнтиДебил - система, выявляющая код, которая позволяет завладеть внутренностями сайта, или уничтожить сайт.
```
- Будет ли нейсколько языков программирования?
```
После создания основы, будет созданна v1 курс по Python. После при изучении новых языков будут созданны новые курсы. Так же каждый пользователь может помочь в разработке курса
```
- Можно ли будет взять основу сайта и использовать для себя?
```
Да! Только нужно будет поменять ребрейдинг и добавить свои курсы
```
- Будет ли панель админа для легкой настройки и создания курсов?
```
Да! Но совсем позже. Пока что будет создание курсов напрямую в базу данных.
```
